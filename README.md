# Employee Management System (Java)

## 📌 Overview
This is a desktop application built using Java Swing to manage employees, departments, and payroll.

## 🚀 Features
- Add, Update, Delete Employees
- Department Management
- Payroll Processing
- Payslip Generation

## 🛠️ Tech Stack
- Java
- Swing
- SQL / JavaDB



## ▶️ How to Run
1. Clone the repository
2. Open in NetBeans or IntelliJ
3. Run the project
